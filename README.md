# סימולטור החלטות: קו דק

משחק ניהול חיים ארוך בעברית.
אתה מנהל עסק, השקעות, בית וקשר עם מאיה.
כל שבוע יש החלטה חדשה, וכל חודש יש סיכום כספי מלא.

## איך מריצים
1. פותחים את `index.html` בדפדפן.
2. אין צורך בשרת או ספריות חיצוניות.

## פריסה ל-GitHub Pages
1. מעלים את הקבצים הבאים לריפו:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
2. ב-GitHub נכנסים ל-`Settings` -> `Pages`.
3. בוחרים `Deploy from branch`.
4. בוחרים ענף `main` ותיקייה `/ (root)`.

## מה חדש במשחק
- משחק ארוך: 60–120 תורות עד סיום רגיל.
- זמן מתקדם לפי שבוע/חודש.
- מחזור חודשי עם הכנסות, החזרים, הוצאות בית ורווחי השקעה.
- מערכת רכוש: קנייה/מכירה והשפעה חודשית.
- מערכת קשרים: זוגיות, חתונה, ילדים, רומן מהצד, חשד בבית.
- מערכת רמזים רחבה שמשפיעה על מסלולים וסיומות.
- 20 סיומות + באג'ים לאיסוף.
- מצב `המשך חופשי+` אחרי פתיחת סיום ראשון.

## טאבים בממשק
- `סיפור`: ההחלטה השבועית והתוצאה.
- `סטטיסטיקות`: כל המדדים בזמן אמת.
- `רכוש`: מה בבעלותך ומה אפשר לקנות.
- `קשרים`: מצב הקשר, הבית והלחץ האישי.
- `רמזים`: כל הרמזים שנאספו.
- `סיומות`: מה נפתח + באג'ים.

## איפה עורכים תוכן
כל התוכן נמצא ב-`app.js`:
- תרחישים: `SCENARIOS` ו-`EXTRA_SCENARIOS`
- עסקאות: `DEALS`
- אירועי צד: `RANDOM_EVENTS`
- רמזים: `CLUES`
- סיומות: `ENDINGS`
- רכוש: `ITEM_CATALOG`

## שמירה מקומית
המשחק נשמר אוטומטית ב-`localStorage`:
- מצב ריצה נוכחי
- סיומות שנפתחו
- באג'ים
- מצב עיצוב (כהה/בהיר)
