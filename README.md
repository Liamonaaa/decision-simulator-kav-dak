# סימולטור החלטות: קו דק

משחק סימולציית חיים ארוך בעברית (RTL).

אתה מנהל כמה חזיתות ביחד:
- עסק והשקעות
- כסף, חובות ותזרים
- בית ומשפחה
- קשרים, מתח רומנטי, סיכון להיתפס

## מה חדש בגרסה הזו

- שמות דמויות רנדומליים בכל משחק חדש:
  - דמות מרכזית לקשר
  - בן/בת זוג רשמי/ת
  - שותף/ה עסקי/ת
  - קרוב/ת משפחה
- טאב `פעולות` קבוע:
  - בוחרים 1–3 פעולות כל שבוע
  - ואז לוחצים `סיים שבוע`
- מערכת חשד חבויה (0–100):
  - עולה מרומן, פעולות לילה, חשיפה גבוהה, קשר גבוה עם אמון נמוך
  - יכולה להפעיל עימות, פרידה/גירושין, דליפה או סחיטה
- בניית משפחה:
  - מעבר לגור יחד, נישואין, ילד/ים
  - לכל ילד יש מחיר חודשי והשפעה על אנרגיה/לחץ/בית
- רכוש (מיני RPG):
  - קנייה ומכירה של פריטים (רכב, דירה, עסק וכו')
  - כל פריט משפיע בכל חודש
- מחזור זמן:
  - כל תור = שבוע
  - כל 4 שבועות = סיכום חודש עם הכנסות/הוצאות
- סיומות מאוחרות:
  - סיומות רגילות רק אחרי 80+ שבועות
  - סיום מוקדם קורה רק בקטסטרופה
- `המשך חופשי+` אחרי פתיחת סיום ראשון

## איך משחקים

1. נכנסים לטאב `פעולות`.
2. בוחרים 1–3 פעולות לשבוע.
3. אפשר גם לבחור תגובה בטאב `סיפור`.
4. לוחצים `סיים שבוע`.
5. קוראים תוצאה וממשיכים לשבוע הבא.

## טאבים

- `סיפור`: מצב שבועי + תגובת סיפור + תוצאה
- `פעולות`: שליטה מלאה על החיים שלך
- `רכוש`: קנייה/מכירה והשפעה חודשית
- `קשרים`: סטטוס אישי + היסטוריית החלטות
- `משפחה`: מצב בית, ילדים, פעולות משפחה
- `רמזים`: רמזים שנאספו
- `סיומות`: אוסף סיומות ובאדג׳ים

## הרצה מקומית

פשוט פותחים את `index.html` בדפדפן.

אין צורך בספריות או שרת צד־שרת.

## פריסה ל-GitHub Pages

1. דוחפים את הקבצים לריפו:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
2. ב-GitHub: `Settings -> Pages`
3. בוחרים:
   - `Deploy from branch`
   - `main`
   - `/ (root)`

## איפה עורכים תוכן

הכול בקובץ `app.js`:
- מערכי שמות: `NAMES`
- פעולות שחקן: `ACTIONS`
- אירועי סיפור: `BEATS`
- אירועים רנדומליים: `EVENTS`
- רמזים: `CLUES`
- סיומות: `ENDINGS`
- רכוש: `ITEMS`

## שמירה

המשחק נשמר אוטומטית ב-`localStorage`:
- ריצה נוכחית
- סיומות ובאדג׳ים שנפתחו
- ערכת צבע
- מצב שפה פשוטה
