# סימולטור החלטות: קו דק

משחק סימולציית חיים בסגנון BitLife בעברית (RTL).

אתם מתחילים בגיל 18.
כל לחיצה על `התקדם שנה` מייצרת כמה אירועים קצרים עם בחירות.
אחרי כל בחירה רואים מיד את ההשלכות.

## מה יש במשחק

- ציר זמן לפי גיל ושנה
- מהירות שנה לבחירה:
  - איטי (3 אירועים)
  - רגיל (5 אירועים)
  - מטורף (8 אירועים)
- טאבים קבועים:
  - חיים
  - עבודה
  - אהבה
  - משפחה
  - נכסים
  - בריאות
  - לימודים
  - עסקים
  - השקעות
  - יומן
- מנוע אירועים גדול (250 תבניות אירוע)
- דמויות עם שמות רנדומליים בכל משחק חדש
- מערכת קשרים, חשד, חשיפה ורומן (מרומז, לא גרפי)
- סיכום שנה ברור:
  - כסף התחיל/הסתיים
  - עבודה
  - אהבה
  - משפחה
  - בריאות
  - 3 אירועים מרכזיים
- 25 סיומות + אוסף סיומות שנפתחו
- New Game Plus עם בונוס פתיחה אחרי פתיחת סיום ראשון

## איך מריצים מקומית

פשוט פותחים את `index.html` בדפדפן.

אין צורך בשרת או חבילות.

## פריסה ל-GitHub Pages

1. דוחפים את הקבצים ל־`main`
2. נכנסים ל־`Settings -> Pages`
3. בוחרים:
   - `Deploy from branch`
   - `main`
   - `/ (root)`

## איפה עורכים תוכן במשחק

כל הלוגיקה ב־`app.js`:

- `NAME_POOLS` – מאגר שמות לדמויות
- `EVENT_PARTS` + `buildEventTemplates` – תבניות אירועים
- `ENDINGS` – סיומות ותנאים
- `ACHIEVEMENTS` – הישגים
- `ASSET_CATALOG` – נכסים
- `buildActionDefinitions` + `runActionLogic` – פעולות יזומות
- `startAdvanceYear` / `finishYear` – לולאת שנה וסיכום

## שמירה

המשחק נשמר אוטומטית ב־`localStorage`:

- מצב ריצה נוכחי (גיל, שנה, סטטיסטיקות, דמויות, תור אירועים, יומן)
- מטא־נתונים (סיומות שנפתחו, הישגים, ערכת צבע, מהירות שנה)
