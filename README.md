# סימולטור החלטות: קו דק

משחק החלטות אינטראקטיבי בעברית (RTL), מבוסס דפדפן בלבד.
העלילה משלבת ביזנס, השקעות, לחץ כספי, משפחה ומתח אישי.

## מה יש כאן
- מנוע החלטות עם הסתעפויות ותוצאות מיידיות ומושהות.
- 30 תרחישים, אירועי צד אקראיים, רמזים וסיומות נפתחות.
- שמירה מלאה ב־`localStorage` כולל ריצה פעילה, הגדרות ופרוגרס סיומות.
- בנוי מ־HTML/CSS/JavaScript בלבד, ללא ספריות חיצוניות.

## הרצה מקומית
1. פתח את הקובץ `index.html` בדפדפן.
2. אפשר גם להריץ שרת סטטי פשוט אם רוצים בדיקות נוחות יותר.

## פריסה ל־GitHub Pages
1. העלה את הקבצים `index.html`, `styles.css`, `app.js`, `README.md` לריפו.
2. ב־GitHub עבור אל `Settings` > `Pages`.
3. בחר מקור פריסה מתוך הענף הרלוונטי (למשל `main` / root).
4. שמור והמתן ליצירת כתובת האתר.

## עריכת תוכן המשחק ב־app.js
- תרחישים: במערך `SCENARIOS`.
- רמזים: במערך `CLUES`.
- סיומות: במערך `ENDINGS`.
- אירועי צד: במערך `RANDOM_EVENTS`.
- תנאי סיום גלובליים: בפונקציה `checkEnding`.
- ערכי פתיחה: באובייקט `INITIAL_STATS`.

## איפוס שמירה
- התחלה חדשה מוחקת ריצה פעילה ושומרת סיומות שנפתחו.
- ניקוי מלא אפשרי דרך ניקוי נתוני אתר בדפדפן (`localStorage`).
