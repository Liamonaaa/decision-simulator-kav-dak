# סימולטור החלטות: קו דק

זה משחק סימולציית חיים ארוך בעברית (RTL).

אתה מנהל חיים שלמים:
- עבודה וקריירה
- עסק והשקעות
- כסף, חובות ונכסים
- זוגיות, משפחה וילדים
- סודות, חשד, חשיפה ורמזים

כל שבוע יש 3 נקודות פעולה.
אתה בוחר פעולות, מגיב למצב שבועי, ואז לוחץ `סיים שבוע`.

## מה יש במשחק עכשיו

- שמות דמויות רנדומליים בכל משחק חדש:
  - דמות רומנטית
  - בן/בת זוג רשמי/ת
  - שותף/ה עסקי/ת
  - קרוב/ת משפחה
- מערכת שבועית ארוכה (60–120+ שבועות)
- סיכום חודשי כל 4 שבועות:
  - שכר, עסק, השקעות
  - הוצאות, חובות, תחזוקת נכסים
- טאבים קבועים:
  - `סיפור`
  - `פעולות`
  - `רכוש`
  - `קשרים`
  - `משפחה`
  - `רמזים`
  - `סיומות`
- מערכת חשד חבויה לכל דמות
- מערכת רמזים (45+ רמזים)
- 20 סיומות + הישגים
- `המשך חופשי+` אחרי פתיחת סיום ראשון

## איך מריצים מקומית

פותחים את `index.html` בדפדפן.

אין צורך בשרת או ספריות חיצוניות.

## פריסה ל-GitHub Pages

1. מעלים את הקבצים לריפו:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
2. ב-GitHub הולכים ל:
   - `Settings -> Pages`
3. בוחרים:
   - `Deploy from branch`
   - `main`
   - `/ (root)`

## איפה עורכים תוכן

כל התוכן נמצא ב-`app.js`:
- `NAME_POOLS` – שמות רנדומליים
- `ACTIONS` – פעולות שחקן
- `SITUATION_SEEDS` + `SITUATION_CHOICES` – מצבי סיפור שבועיים
- `EVENT_TEXTS` – אירועים רנדומליים (80 טקסטים)
- `CLUE_BANK` – רמזים
- `ENDINGS` – סיומות
- `ACHIEVEMENTS` – הישגים
- `ASSET_CATALOG` – רכוש

## שמירה

המשחק נשמר אוטומטית ב-`localStorage`:
- ריצה נוכחית
- סיומות והישגים שנפתחו
- ערכת צבע
- מצב שפה פשוטה
